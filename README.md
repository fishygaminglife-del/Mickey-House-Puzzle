# Mickey Mouse Club House Puzzle

**Mickey Mouse Club House Puzzle** is a 2D game made in the Godot game engine.  
The game features a light storyline and many interactive puzzles and mini-games for the player to complete.

As players explore the environment, they can interact with objects and complete challenges that transport them to new levels.

---

## Features
- 2D gameplay
- Interactive puzzles and mini-games
- Simple storyline
- Level transitions when puzzles are completed
- Keyboard and mouse controls
